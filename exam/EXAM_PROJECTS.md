## Проекты

### 1. Переговорки

> 😤 *«Я ЗАДОЛБАЛСЯ искать свободную переговорку! Прихожу — занято. Иду к другой — тоже занято. Outlook? Там НИКТО не отмечает когда освободил раньше!»*

**Что нужно**: Telegram-бот. Пишу «свободна?» — показывает свободные переговорки. «Забронируй Марс на 15:00» — бронирует.

**Стек**: `Python` `aiogram` `SQLite`

## Как работать

```
1. Выбери проект (что тебе близко)
2. Создай репозиторий на GitHub
3. Напиши BDD сценарии в .feature файлах
4. Покажи их Claude, попроси сгенерировать код
5. Коммить каждые 15-20 минут
6. Напиши README и CLAUDE.md
7. Убедись что тесты проходят
8. Отправь ссылку на репозиторий
```

## Структура репозитория

```
my-project/
├── README.md           # Описание + как запустить
├── CLAUDE.md           # Как работал с AI
├── requirements.txt    # Зависимости
├── src/
│   └── ...            # Код
├── tests/
│   ├── features/      # BDD .feature файлы
│   └── steps/         # Step definitions
└── .gitignore
```
